## Жёсткие правила (никогда не нарушать)

- **Data isolation:** UI-компоненты вызывают данные ТОЛЬКО через `lib/services/`. Никогда напрямую из MCP/API.
- **Slug:** единственный ID монеты — `[номинал]-[год]-[название-латиницей]`. Неизменяем после создания.
- **Изображения:** только через `next/image`. Путь: `/public/images/coins/{slug}-obverse.jpg` и `{slug}-reverse.jpg`.
- **TypeScript:** `strict: true`. Никаких `any`. Типы для всех данных.
- **Google Sheets Hard Block:** перед любой работой с MCP — прочитать структуру таблицы, не угадывать имена колонок.
- **Константы:** все строки-идентификаторы в `lib/constants.ts`, никаких magic strings в коде.
- **Коммиты:** после каждого важного изменения — предлагать текст коммита.
